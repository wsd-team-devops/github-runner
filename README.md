This branch is for solvians side self-hostes GitHub runner config setup, we have to use Dockerfile to build up our custom image

Runner version : 2.316.0
ImageOS: ubuntu22

Software installed:
1. Yarn
2. Nodejs
3. Git
4. Self-signed Solvians CA cert
